# 🎯 Process Hunter

**Process Hunter** is a lightweight and powerful C++ utility designed to detect hidden processes and miners in Windows.

---

## ✨ Features
* 🔍 **100% Accuracy**: Displays the real physical file paths.
* 🛡️ **System Protection**: Features a built-in whitelist for critical system processes.
* 🪶 **Lightweight**: Small file size and no heavy external libraries required.
* 🛑 **Kill Feature**: Allows you to instantly terminate any process by its PID.

---

## 🚀 How to Run

1. Download the `Process Hunter.exe` file from the **Releases** section.
2. Run it **as Administrator** (required for full access to file paths).
3. Enter the PID of the suspicious process to terminate it.
